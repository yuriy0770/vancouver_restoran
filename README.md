# Canadian Maple Restaurant - Система бронирования столиков

Веб-приложение для бронирования столиков в ресторане "Canadian Maple" с адаптивным дизайном и полным функционалом управления бронированиями.

## 🚀 Функциональность

### Для гостей
- 📖 Просмотр информации о ресторане
- 🍽️ Бронирование столиков через интуитивную форму
- 📱 Адаптивный дизайн для всех устройств
- 👤 Регистрация и авторизация
- 📋 Просмотр истории бронирований

### Для администраторов
- ⚙️ Полное управление бронированиями через админ-панель
- 👥 Управление пользователями
- 🪑 Управление столиками
- 📊 Просмотр статистики

## 🛠 Технологии

- **Backend**: Django 5.0, Python 3.11
- **Frontend**: Bootstrap 5, HTML5, CSS3
- **Database**: PostgreSQL
- **Контейнеризация**: Docker, Docker Compose
- **Деплой**: Gunicorn (готово для продакшена)
- **Уведомления**: Telegram бот

## 🐛 Troubleshooting
Common Issues
Ошибка подключения к БД
Проверьте настройки PostgreSQL
Убедитесь что база данных создана
Статические файлы не загружаются
Выполните python manage.py collectstatic
Проверьте настройки STATIC_ROOT
Проблемы с Docker
Очистите кэш: docker system prune
Пересоберите образы: docker-compose build --no-cache
